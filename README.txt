5. Cek address adminPool dengan menekan tombol adminPool

6. Cek saldo sesuai addres dengan menekan tombol balanceOf, kemudian memasukkan addressnya

7. isAdmin, untuk mengecek kebenaran address dari Admin

8. isRecyclingPersonnel, untuk mengecek kebenaran address dari Personnel

9. isVendor, untuk mengecek kebenaran address dari Vendor

10. rewardRates, untuk mengecek rate tiap sampah 0=plastic, 1=glass, 2=metal, 3=paper

11. vendorDeposit, untuk menginput saldo voucher melalui Vendor

12. depositTrash, untuk menginput sampah sebagai Personnel

13. processTrash, untuk menginput jumlah sampah yang diproses sebagai Admin

14. setRewardRate, untuk mengeset reward berdasarkan jenis sampah

15. withdrawPayout, untuk menarik saldo voucher kepada investor
